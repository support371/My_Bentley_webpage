# Mobile Execution Steps

1. Keep all implementation work inside the active app tree.
2. Reuse current event, inventory, integration, and governance data.
3. Treat the mobile pages as a bounded module.
4. Keep control surfaces protected by the current auth model.
5. Use lightweight stubs only where persistence does not exist yet.
6. Update smoke coverage whenever a new route is added.
7. Merge in increments instead of attempting a single oversized drop.
