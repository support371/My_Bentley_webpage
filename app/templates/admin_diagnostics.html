curl Test Commands

Copy and replace placeholders

Token request:

curl -s -X POST \ https://ims.bentley.com/connect/token \ -d "grant_type=client_credentials" \ -d "client_id=YOUR_CLIENT_ID" \ -d "client_secret=YOUR_CLIENT_SECRET" \ -d "scope=itwins:read imodels:read webhooks:read webhooks:modify"

iTwins test (replace ACCESS_TOKEN):

curl -s \ https://api.bentley.com/itwins?$top=1 \ -H "Authorization: Bearer ACCESS_TOKEN" \ -H "Accept: application/vnd.bentley.itwin-platform.v1+json"

Webhooks test (replace ACCESS_TOKEN):

curl -s \ https://api.bentley.com/webhooks \ -H "Authorization: Bearer ACCESS_TOKEN" \ -H "Accept: application/vnd.bentley.itwin-platform.v1+json"
